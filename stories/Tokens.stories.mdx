# Design Tokens Documentation

Welcome to the Design Tokens documentation. This Storybook contains all design tokens used in the design system.

## Overview

Our design tokens are organized into three main categories:

1. **Primitives** - Base tokens with raw values (colors, spacing, typography, etc.)
2. **Semantic Tokens** - Meaning-based tokens that reference primitives
3. **Component Tokens** - Component-specific tokens that reference semantic tokens
